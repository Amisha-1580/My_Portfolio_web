🌐 Personal Portfolio Website
📌 Project Overview

This project is a Personal Portfolio Website developed as part of an industrial training program.
It showcases personal details, skills, projects, and contact information in a professional and responsive web format.

The main goal of this project is to build a strong online presence using basic web technologies
and follow proper project structure and version control practices.


🛠️ Technologies Used

HTML5 – Structure of the website

CSS3 – Styling and layout design

JavaScript – Interactivity and dynamic behavior

Git & GitHub – Version control and hosting


✨ Features

Responsive design (mobile, tablet, desktop)

Home, About, Skills, Projects, and Contact sections

Clean and user-friendly UI

Navigation menu with smooth scrolling

Organized folder structure

Ready for GitHub Pages deployment
